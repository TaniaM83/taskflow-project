# TaskFlow

TaskFlow es una app web sencilla para organizar tus tareas del dia a dia. La idea es que puedas apuntar lo que tienes que hacer, ponerle fechas y ver de un vistazo en que estado esta cada cosa.

## Que puedes hacer

- Crear tareas con un nombre, una fecha de inicio y una de fin
- Cambiar el estado de cada tarea entre Pendiente, En proceso y Terminada haciendo clic en el badge
- Ver un calendario que se actualiza mes a mes y muestra puntitos de colores en los dias que tienen tareas
- Filtrar las tareas por estado para encontrar rapido lo que buscas
- Eliminar tareas que ya no necesites
- Tus tareas se guardan en el navegador, asi que no se pierden al refrescar la pagina

## Como esta hecho

El proyecto esta construido solo con HTML, CSS y JavaScript, sin librerias ni frameworks. El diseno usa una paleta de colores verde oliva y es responsive para que se vea bien tanto en el ordenador como en el movil.

## Archivos del proyecto

- `src/index.html` - La estructura de la pagina
- `src/app.js` - Toda la logica: tareas, calendario, localStorage
- `src/styles/style.css` - Los estilos y el diseno responsive
